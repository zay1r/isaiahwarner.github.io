Team LeaderBoard Full Stack WEb application

full-stack web application that served as a leaderboard for the tournament. I created this leaderboard to Track the team scores for a Charity Golf Scramble event. I used a combination of front-end and back-end technologies to build the application. On the front-end, I used HTML, CSS, and JavaScript to create a user-friendly interface that allowed users to view the leaderboard and team scores. On the back-end, I used Node.js, and Render to create a server that handled requests and responses. I also used a Postgres SQL database to store the team scores and other relevant information like Team names and Passwords, and I used it to seed the database with the hole information. The application allowed users to view the leaderboard in real-time, with scores updating automatically as teams completed their rounds.
